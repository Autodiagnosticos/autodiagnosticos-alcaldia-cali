⚙️ Guía de Configuración

Autodiagnósticos en línea con Google Sheets · Alcaldía de Santiago de Cali

1
Crear la hoja de Google Sheets
Donde quedarán registradas todas las respuestas
📊
Abrir Google Sheets y crear una hoja nueva
Ve a sheets.google.com → clic en "+" para crear una hoja nueva → nómbrala Autodiagnósticos Alcaldía de Cali.
🔗
Copiar el ID de la hoja desde la URL
En la barra del navegador verás algo así:
https://docs.google.com/spreadsheets/d/1BxiMVs0XRA5nFMdKvBdBZjgmUUqptlbs74OgVE2upms/edit ↑ ESTE ES EL ID ↑
⚠️ Guarda ese ID, lo necesitarás en el paso siguiente.
📑
Renombrar la primera pestaña
Clic derecho en la pestaña inferior → Cambiar nombre → escribe Respuestas (exactamente así, con mayúscula).
2
Instalar el Google Apps Script
El código que recibe los formularios y escribe en Sheets
🛠️
Abrir el editor de Apps Script
Desde tu Google Sheet: menú Extensiones → Apps Script. Se abre una pestaña nueva.
📋
Pegar el código
Borrar todo lo que hay → abrir el archivo APPS_SCRIPT_SHEETS.js que está en tu carpeta → copiar todo el contenido → pegar en el editor de Apps Script.
✏️
Reemplazar el ID de la hoja
En la línea 35 del código, reemplaza el texto entre comillas con tu ID real:
// Antes: const SHEET_ID = 'PEGA_AQUI_EL_ID_DE_TU_GOOGLE_SHEET'; // Después (ejemplo): const SHEET_ID = '1BxiMVs0XRA5nFMdKvBdBZjgmUUqptlbs74OgVE2upms';
💾
Guardar el proyecto
Ctrl+S o clic en el ícono de guardar 💾. Ponle un nombre al proyecto cuando lo pida (ej: Receptor Autodiagnósticos).
🚀
Publicar como Aplicación Web
Clic en Implementar → Nueva implementación:

• Tipo: Aplicación web
• Descripción: Autodiagnósticos Alcaldía Cali
• Ejecutar como: Yo (tu correo)
• Quién puede acceder: Cualquier usuario

→ Clic en Implementar → Autorizar cuando lo pida → Copiar la URL de la aplicación web.
✅ La URL se verá así: https://script.google.com/macros/s/AKfycb.../exec
3
Conectar los formularios con tu Sheets
Reemplazar la URL en los 15 archivos HTML
🔍
Buscar el texto a reemplazar en cada HTML
En cada uno de los 15 archivos AUTODIAGNOSTICO_*.html, busca (Ctrl+F en un editor de texto):
REEMPLAZAR_CON_URL_DE_APPS_SCRIPT
✏️
Reemplazar con tu URL real
Pega la URL que copiaste en el paso anterior. Ejemplo:
// Antes: const SCRIPT_URL = 'REEMPLAZAR_CON_URL_DE_APPS_SCRIPT'; // Después: const SCRIPT_URL = 'https://script.google.com/macros/s/AKfycbXXXXXXX/exec';
⚠️ Consejo: usa Notepad++ o VS Code para hacer el reemplazo en todos los archivos a la vez (Ctrl+Shift+H → Reemplazar en archivos).
4
Publicar los archivos en línea
Para que se puedan diligenciar desde cualquier equipo
🌐
Opción A: Servidor web institucional (recomendado)
Sube todos los archivos HTML a la carpeta del servidor web de la Alcaldía. El administrador de TI puede ayudarte. Los archivos funcionan sin ningún backend adicional.
📂
Opción B: GitHub Pages (gratis y sin servidor)
1. Crear cuenta en github.com
2. Nuevo repositorio → subir todos los archivos HTML
3. Settings → Pages → Source: main branch
4. Tu URL quedará: https://tuusuario.github.io/autodiagnosticos/
💡
Opción C: Netlify Drop (más fácil)
Ir a app.netlify.com/drop → arrastrar la carpeta completa → en segundos tienes una URL pública con todos los archivos. Gratis.
✅ Una vez publicados, comparte el enlace de autodiagnosticos.html y desde ahí los funcionarios acceden a todos los formularios.
